Déscription


Ce site est pour l'examen de Stradji. Il a été réalisé le 30 avril 2016 par Ayoub EL Asry. Le site permet d'afficher des photos en fonction de la recherche. Pour ce faire une requête est envoyée au site Flickr qui retourne un format JSON de la recherche entrée. Par la suite, le format JSON est traduit en code HTML.

Contenu

Le site utilise plusieurs technologies web.
En effet, ce site utilise des fonctions pour ajuster la taille de certains éléments, afficher du contenu et modifier un contenu.
Il gère aussi des éléments à l extérieur du site comme les requêtes sur Flickr.
Pour la partie design, bootsrap permet d'avoir un des layouts qui s'adapte au différend tailles d'écran.

Technologies

Plus en détail, dans la page recherche le JavaScript sert à envoyer une sorte de requête a Flickr. Par la suite en fonction traite le retour de cette requête.
JQuery quant à lui est plus utilisé pour les animations.
Le CSS charge le design de la page. Le HTML sert à ordonner les éléments.

Liste des technologies utilisées
  • HTML5 et CSS
  • JavaScript et JQuery
  • PHP
  • Bootstrap
